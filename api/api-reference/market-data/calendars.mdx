## Get Trading Calendars

Retrieve trading calendar information, including market holidays, special sessions, and trading schedule changes.

### Endpoint
```http
GET /api/v1/market/calendars
```

### Parameters

<ParamField query="year" type="integer">
  Year to retrieve calendar for (e.g., 2024). Defaults to current year.
</ParamField>

<ParamField query="month" type="integer">
  Month to filter (1-12). If not provided, returns entire year.
</ParamField>

<ParamField query="instrument" type="string">
  Filter for specific instrument calendar (e.g., "SP500-BTC")
</ParamField>

<ParamField header="X-API-Key" type="string" required>
  API key for authentication
</ParamField>

### Example Request

<RequestExample>
```bash Current Year
curl -X GET "https://api.roxom.com/api/v1/market/calendars" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```bash Specific Month
curl -X GET "https://api.roxom.com/api/v1/market/calendars?year=2024&month=12" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```python Python
import requests
from datetime import datetime

headers = {
    'X-API-Key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
}

# Get current year calendar
current_year = datetime.now().year
response = requests.get(
    'https://api.roxom.com/api/v1/market/calendars',
    headers=headers,
    params={'year': current_year}
)

calendar_data = response.json()
for event in calendar_data['data']:
    print(f"{event['date']}: {event['name']} - {event['type']}")
```

```javascript JavaScript
const response = await fetch('https://api.roxom.com/api/v1/market/calendars?year=2024', {
  headers: {
    'X-API-Key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const calendar = await response.json();
calendar.data.forEach(event => {
  console.log(`${event.date}: ${event.name} (${event.type})`);
});
```
</RequestExample>

## Understanding Trading Calendars

<AccordionGroup>
  <Accordion title="Market Holidays">
    Trading may be affected by holidays in the underlying markets:
    - **US Holidays**: Affect SP500-BTC (follows NYSE calendar)
    - **Global Markets**: GLD-BTC and OIL-BTC may have different schedules
    - **24/7 Trading**: Bitcoin settlement continues during holidays
    - **Reduced Liquidity**: Expect lower volume during holiday periods
  </Accordion>

  <Accordion title="Event Types">
    - **Holiday**: Full market closure, no trading
    - **Early Close**: Market closes earlier than usual
    - **Late Open**: Market opens later than usual  
    - **Special Session**: Modified trading rules or hours
  </Accordion>

  <Accordion title="Impact on Trading">
    - **Position Management**: Plan exits before holidays
    - **Funding Payments**: May be adjusted during closures
    - **Order Books**: Reduced liquidity during affected periods
    - **Risk Management**: Higher volatility around closures
  </Accordion>

  <Accordion title="Instrument-Specific Rules">
    Different instruments may have different holiday schedules:
    - **SP500-BTC**: Follows US market holidays
    - **GLD-BTC**: Follows precious metals market schedule
    - **OIL-BTC**: Follows energy market schedule
    - **Bitcoin Settlement**: Always available 24/7
  </Accordion>
</AccordionGroup>

<Info>
While underlying markets may close, Bitcoin settlement and basic platform operations continue 24/7. However, reduced liquidity is expected during holiday periods.
</Info>

<Warning>
Always check the calendar before major holidays to plan your trading strategy and position management accordingly.
</Warning>
