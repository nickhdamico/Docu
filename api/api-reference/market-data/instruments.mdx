## Get Trading Instruments

Retrieve information about all available trading instruments, including contract specifications and trading rules.

### Endpoint
```http
GET /api/v1/market/instruments
```

### Parameters

<ParamField query="symbol" type="string">
  Filter for specific trading pair (e.g., "SP500-BTC"). If not provided, returns all instruments.
</ParamField>

<ParamField query="status" type="string">
  Filter by trading status: "active", "suspended", "delisted"
</ParamField>

<ParamField header="X-API-Key" type="string" required>
  API key for authentication
</ParamField>

### Example Request

<RequestExample>
```bash All Instruments
curl -X GET "https://api.roxom.com/api/v1/market/instruments" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```bash Single Instrument
curl -X GET "https://api.roxom.com/api/v1/market/instruments?symbol=SP500-BTC" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```python Python
import requests

headers = {
    'X-API-Key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
}

# Get all active instruments
response = requests.get(
    'https://api.roxom.com/api/v1/market/instruments',
    headers=headers,
    params={'status': 'active'}
)

instruments = response.json()
for instrument in instruments['data']:
    print(f"Symbol: {instrument['symbol']}")
    print(f"Min order size: {instrument['min_order_size']} satoshis")
```

```javascript JavaScript
const response = await fetch('https://api.roxom.com/api/v1/market/instruments?status=active', {
  headers: {
    'X-API-Key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const instruments = await response.json();
instruments.data.forEach(instrument => {
  console.log(`${instrument.symbol}: ${instrument.description}`);
});
```
</RequestExample>

## Understanding Trading Instruments

<AccordionGroup>
  <Accordion title="Available Instruments">
    Roxom currently offers three main trading instruments:
    - **SP500-BTC**: S&P 500 Index futures
    - **GLD-BTC**: Gold ETF futures  
    - **OIL-BTC**: Crude Oil futures
    
    All instruments are settled in Bitcoin (BTC).
  </Accordion>

  <Accordion title="Contract Specifications">
    - **Perpetual Contracts**: No expiry date, trade indefinitely
    - **Bitcoin Settlement**: All profits/losses in BTC
    - **Funding Payments**: Every 8 hours to maintain price peg
    - **24/7 Trading**: Continuous trading with no market close
  </Accordion>

  <Accordion title="Size and Pricing">
    - **Minimum Sizes**: Vary by instrument for optimal liquidity
    - **Tick Sizes**: 1 satoshi minimum price increment
    - **Leverage**: Up to 12x depending on instrument
    - **Margin**: Initial margin requirements 8-12%
  </Accordion>

  <Accordion title="Fee Structure">
    - **Maker Fees**: 0.02-0.03% for adding liquidity
    - **Taker Fees**: 0.05-0.06% for removing liquidity
    - **Funding Fees**: Variable based on market conditions
    - **No Hidden Fees**: Transparent fee structure
  </Accordion>
</AccordionGroup>

<Tip>
Use this endpoint to check current trading rules and specifications before placing orders, as parameters may be updated periodically.
</Tip>
