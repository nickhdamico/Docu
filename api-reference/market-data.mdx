## Overview

The Market Data API provides access to real-time quotes, historical price data, orderbook information, and trading statistics for all supported instruments.

## Get Current Price

Retrieve the current market price for a specific trading pair.

### Endpoint
```http
GET /v1/market/price/{symbol}
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `symbol` | string | Yes | Trading pair symbol (e.g., "BTC-USD") |

### Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.roxom.io/v1/market/price/BTC-USD" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://api.roxom.io/v1/market/price/BTC-USD',
    headers=headers
)

price_data = response.json()
print(f"Current price: {price_data['data']['price']}")
```

```javascript JavaScript
const response = await fetch('https://api.roxom.io/v1/market/price/BTC-USD', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const priceData = await response.json();
console.log('Current price:', priceData.data.price);
```
</CodeGroup>

### Response

```json

  "data": {
    "symbol": "BTC-USD",
    "price": "45127.68000000",

    "ask": "45129.86000000",

  },
  "timestamp": "2024-01-15T10:30:00Z",

}
```

## Get Order Book

Retrieve current order book data for a trading pair.

### Endpoint
```http
GET /v1/market/orderbook/{symbol}
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `symbol` | string | Yes | Trading pair symbol |
| `depth` | integer | No | Number of price levels (default: 20, max: 100) |

### Example Request

```bash
curl -X GET "https://api.roxom.io/v1/market/orderbook/BTC-USD?depth=10" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

### Response

```json
{
  "data": {
    "symbol": "BTC-USD",
    "bids": [
      ["45125.50000000", "0.25000000"],
      ["45125.00000000", "0.50000000"],
      ["45124.50000000", "1.00000000"]
    ],
    "asks": [
      ["45129.86000000", "0.30000000"],
      ["45130.00000000", "0.75000000"],
      ["45130.50000000", "0.45000000"]
    ],
    "timestamp": "2024-01-15T10:30:00Z"
  }
}
```

## Get 24hr Statistics

Retrieve 24-hour trading statistics for one or all trading pairs.

### Endpoint
```http
GET /v1/market/stats/{symbol}
GET /v1/market/stats
```

### Example Request

<CodeGroup>
```bash Single Symbol

  -H "Authorization: Bearer YOUR_API_KEY" \
```

## Get Historical Klines

Retrieve historical kline/candlestick data.

### Endpoint
```http
GET /v1/market/klines/{symbol}
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `symbol` | string | Yes | Trading pair symbol |
| `interval` | string | Yes | Kline interval (1m, 5m, 15m, 30m, 1h, 4h, 1d, 1w) |
| `start_time` | integer | No | Start timestamp (Unix milliseconds) |
| `end_time` | integer | No | End timestamp (Unix milliseconds) |
| `limit` | integer | No | Number of klines (default: 500, max: 1000) |

### Example Request

```bash
curl -X GET "https://api.roxom.io/v1/market/klines/BTC-USD?interval=1h&limit=24" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

### Response

```json
{
  "data": [
    [
      1642248600000,           // Open time
      "44950.00000000",        // Open price
      "45050.00000000",        // High price
      "44900.00000000",        // Low price
      "45000.00000000",        // Close price
      "125.45000000",          // Volume
      1642252199999,           // Close time
      "5641253.45000000",      // Quote asset volume
      856,                     // Number of trades
      "62.75000000",           // Taker buy base asset volume
      "2820626.72500000"       // Taker buy quote asset volume
    ]
  ]
}
```

## Get Recent Trades

Retrieve recent trade executions.

### Endpoint
```http
GET /v1/market/trades/{symbol}
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `symbol` | string | Yes | Trading pair symbol |
| `limit` | integer | No | Number of trades (default: 100, max: 500) |

### Example Request

```bash
curl -X GET "https://api.roxom.io/v1/market/trades/BTC-USD?limit=50" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

### Response

```json
{
  "data": [
    {
      "id": "trade_12345",
      "price": "45127.68000000",
      "quantity": "0.25000000",
      "quote_quantity": "11281.92000000",
      "timestamp": "2024-01-15T10:30:00Z",
      "is_buyer_maker": false
    }
  ]
}
```

## Error Handling

Common error responses for market data endpoints:

### Symbol Not Found

```json
{
  "error": {
    "code": 404,
    "message": "Symbol not found",
    "details": "Trading pair 'INVALID-PAIR' does not exist"
  }
}
```

### Invalid Parameters

```json
{
  "error": {
    "code": 400,
    "message": "Invalid interval",
    "details": "Interval must be one of: 1m, 5m, 15m, 30m, 1h, 4h, 1d, 1w"
  }
}
```

## Rate Limits

Market data endpoints have specific rate limits:

| Endpoint | Rate Limit | Burst Limit |
|----------|------------|-------------|
| Price data | 100/min | 20/sec |
| Order book | 50/min | 10/sec |
| Historical data | 20/min | 5/sec |
| Trade history | 30/min | 6/sec |
