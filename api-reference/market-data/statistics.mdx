## Get 24hr Statistics

Retrieve 24-hour trading statistics for one or all trading pairs.

### Endpoint
```http
GET /v1/market/stats/{symbol}
GET /v1/market/stats
```

### Parameters

<ParamField path="symbol" type="string">
  Trading pair symbol. Omit to get stats for all pairs
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Format: `Bearer YOUR_API_KEY`
</ParamField>

### Example Request

<RequestExample>
```bash Single Symbol
curl -X GET "https://api.roxom.com/v1/market/stats/BTC-USD" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```bash All Symbols
curl -X GET "https://api.roxom.com/v1/market/stats" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

# Single symbol
response = requests.get(
    'https://api.roxom.com/v1/market/stats/BTC-USD',
    headers=headers
)

stats = response.json()
print(f"24h change: {stats['data']['price_change_percent']}%")
```

```javascript JavaScript
// Single symbol
const response = await fetch('https://api.roxom.com/v1/market/stats/BTC-USD', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const stats = await response.json();
console.log('24h change:', stats.data.price_change_percent + '%');
```
</RequestExample>

## Understanding Statistics

<AccordionGroup>
  <Accordion title="Volume Metrics">
    **Base Volume**: Amount of base asset traded (e.g., BTC in BTC-USD)
    
    **Quote Volume**: Amount of quote asset traded (e.g., USD in BTC-USD)
    
    **Trade Count**: Total number of individual trades executed
  </Accordion>

  <Accordion title="Price Metrics">
    **OHLC**: Open, High, Low, Close prices for the 24-hour period
    
    **Price Change**: Absolute difference between current and opening price
    
    **Percentage Change**: Relative change as a percentage of opening price
  </Accordion>

  <Accordion title="Use Cases">
    - **Market Analysis**: Identify trending pairs and market sentiment
    - **Volatility Assessment**: Gauge price stability using high/low spread
    - **Liquidity Evaluation**: Use volume data to assess trading activity
  </Accordion>
</AccordionGroup>
