## Get Historical Klines

Retrieve historical kline/candlestick data for chart analysis.

### Endpoint
```http
GET /v1/market/klines/{symbol}
```

### Parameters

<ParamField path="symbol" type="string" required>
  Trading pair symbol (e.g., "BTC-USD")
</ParamField>

<ParamField query="interval" type="string" required>
  Kline interval: "1m", "5m", "15m", "30m", "1h", "4h", "1d", "1w"
</ParamField>

<ParamField query="start_time" type="integer">
  Start timestamp (Unix milliseconds)
</ParamField>

<ParamField query="end_time" type="integer">
  End timestamp (Unix milliseconds)
</ParamField>

<ParamField query="limit" type="integer" default="500">
  Number of klines to return (max: 1000)
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Format: `Bearer YOUR_API_KEY`
</ParamField>

### Example Request

<RequestExample>
```bash cURL
curl -X GET "https://api.roxom.com/v1/market/klines/BTC-USD?interval=1h&limit=24" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```python Python
import requests
from datetime import datetime, timedelta

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

# Get last 24 hours of hourly data
end_time = int(datetime.now().timestamp() * 1000)
start_time = int((datetime.now() - timedelta(hours=24)).timestamp() * 1000)

response = requests.get(
    'https://api.roxom.com/v1/market/klines/BTC-USD',
    headers=headers,
    params={
        'interval': '1h',
        'start_time': start_time,
        'end_time': end_time
    }
)

klines = response.json()
for kline in klines['data']:
    print(f"Time: {kline[0]}, Open: {kline[1]}, Close: {kline[4]}")
```

```javascript JavaScript
const response = await fetch('https://api.roxom.com/v1/market/klines/BTC-USD?interval=1h&limit=24', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const klines = await response.json();
klines.data.forEach(kline => {
  console.log(`Time: ${new Date(kline[0])}, Open: ${kline[1]}, Close: ${kline[4]}`);
});
```
</RequestExample>

## Kline Data Format

Each kline is an array with the following structure:

<ResponseField name="kline_array" type="array">
  Kline data array format

  <Expandable title="Kline array structure">
    <ResponseField name="[0]" type="integer">
      Open time (Unix timestamp in milliseconds)
    </ResponseField>
    
    <ResponseField name="[1]" type="string">
      Open price
    </ResponseField>
    
    <ResponseField name="[2]" type="string">
      High price
    </ResponseField>
    
    <ResponseField name="[3]" type="string">
      Low price
    </ResponseField>
    
    <ResponseField name="[4]" type="string">
      Close price
    </ResponseField>
    
    <ResponseField name="[5]" type="string">
      Volume (base asset)
    </ResponseField>
    
    <ResponseField name="[6]" type="integer">
      Close time (Unix timestamp in milliseconds)
    </ResponseField>
    
    <ResponseField name="[7]" type="string">
      Quote asset volume
    </ResponseField>
    
    <ResponseField name="[8]" type="integer">
      Number of trades
    </ResponseField>
    
    <ResponseField name="[9]" type="string">
      Taker buy base asset volume
    </ResponseField>
    
    <ResponseField name="[10]" type="string">
      Taker buy quote asset volume
    </ResponseField>
  </Expandable>
</ResponseField>

## Get Recent Trades

Retrieve recent trade executions for a trading pair.

### Endpoint
```http
GET /v1/market/trades/{symbol}
```

### Parameters

<ParamField path="symbol" type="string" required>
  Trading pair symbol (e.g., "BTC-USD")
</ParamField>

<ParamField query="limit" type="integer" default="100">
  Number of trades to return (max: 500)
</ParamField>

### Example Request

<RequestExample>
```bash cURL
curl -X GET "https://api.roxom.com/v1/market/trades/BTC-USD?limit=50" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```
</RequestExample>

## Available Intervals

| Interval | Description | Use Case |
|----------|-------------|----------|
| `1m` | 1 minute | Scalping, high-frequency analysis |
| `5m` | 5 minutes | Short-term trading |
| `15m` | 15 minutes | Intraday analysis |
| `30m` | 30 minutes | Medium-term patterns |
| `1h` | 1 hour | Hourly trends |
| `4h` | 4 hours | Daily trading decisions |
| `1d` | 1 day | Long-term analysis |
| `1w` | 1 week | Weekly trends |
