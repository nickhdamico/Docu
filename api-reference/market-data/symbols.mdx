## Get Trading Symbols

Retrieve a list of all supported trading pairs and their trading rules.

### Endpoint
```http
GET /v1/market/symbols
```

### Parameters

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Format: `Bearer YOUR_API_KEY`
</ParamField>

<ParamField query="symbol" type="string">
  Filter for specific symbol information
</ParamField>

### Example Request

<RequestExample>
```bash All Symbols
curl -X GET "https://api.roxom.com/v1/market/symbols" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```bash Single Symbol
curl -X GET "https://api.roxom.com/v1/market/symbols?symbol=BTC-USD" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://api.roxom.com/v1/market/symbols',
    headers=headers
)

symbols = response.json()
for symbol in symbols['data']:
    print(f"{symbol['symbol']}: {symbol['status']}")
```

```javascript JavaScript
const response = await fetch('https://api.roxom.com/v1/market/symbols', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const symbols = await response.json();
symbols.data.forEach(symbol => {
  console.log(`${symbol.symbol}: ${symbol.status}`);
});
```
</RequestExample>

## Trading Status Values

| Status | Description |
|--------|-------------|
| `trading` | Normal trading is active |
| `halt` | Trading is temporarily halted |
| `break` | Trading break (scheduled maintenance) |
| `maintenance` | Symbol under maintenance |

## Filter Types

<AccordionGroup>
  <Accordion title="PRICE_FILTER">
    Controls valid price ranges and increments:
    
    - **min_price**: Minimum allowed price
    - **max_price**: Maximum allowed price  
    - **tick_size**: Price must be divisible by this value
    
    ```json
    {
      "type": "PRICE_FILTER",
      "min_price": "0.01000000",
      "max_price": "100000.00000000", 
      "tick_size": "0.01000000"
    }
    ```
  </Accordion>

  <Accordion title="LOT_SIZE">
    Controls valid quantity ranges and increments:
    
    - **min_qty**: Minimum order quantity
    - **max_qty**: Maximum order quantity
    - **step_size**: Quantity must be divisible by this value
    
    ```json
    {
      "type": "LOT_SIZE",
      "min_qty": "0.00001000",
      "max_qty": "10000.00000000",
      "step_size": "0.00001000"
    }
    ```
  </Accordion>

  <Accordion title="MIN_NOTIONAL">
    Minimum notional value requirement:
    
    - **min_notional**: Minimum order value (price × quantity)
    
    ```json
    {
      "type": "MIN_NOTIONAL",
      "min_notional": "10.00000000"
    }
    ```
  </Accordion>

  <Accordion title="MAX_NUM_ORDERS">
    Maximum number of open orders per symbol:
    
    - **max_num_orders**: Maximum open orders allowed
    
    ```json
    {
      "type": "MAX_NUM_ORDERS",
      "max_num_orders": 200
    }
    ```
  </Accordion>
</AccordionGroup>

## Order Validation

Use symbol information to validate orders before submission:

```python
def validate_order(symbol_info, price, quantity):
    """Validate order parameters against symbol filters."""
    
    filters = {f['type']: f for f in symbol_info['filters']}
    
    # Check price filter
    if 'PRICE_FILTER' in filters:
        pf = filters['PRICE_FILTER']
        min_price = float(pf['min_price'])
        max_price = float(pf['max_price'])
        tick_size = float(pf['tick_size'])
        
        if not (min_price <= price <= max_price):
            raise ValueError(f"Price {price} outside range [{min_price}, {max_price}]")
            
        if price % tick_size != 0:
            raise ValueError(f"Price {price} not divisible by tick size {tick_size}")
    
    # Check lot size
    if 'LOT_SIZE' in filters:
        ls = filters['LOT_SIZE']
        min_qty = float(ls['min_qty'])
        max_qty = float(ls['max_qty'])
        step_size = float(ls['step_size'])
        
        if not (min_qty <= quantity <= max_qty):
            raise ValueError(f"Quantity {quantity} outside range [{min_qty}, {max_qty}]")
            
        if quantity % step_size != 0:
            raise ValueError(f"Quantity {quantity} not divisible by step size {step_size}")
    
    # Check minimum notional
    if 'MIN_NOTIONAL' in filters:
        min_notional = float(filters['MIN_NOTIONAL']['min_notional'])
        notional = price * quantity
        
        if notional < min_notional:
            raise ValueError(f"Notional {notional} below minimum {min_notional}")
    
    return True
```

## Common Symbol Categories

| Category | Examples | Description |
|----------|----------|-------------|
| **Major Crypto** | BTC-USD, ETH-USD | Large-cap cryptocurrencies vs USD |
| **Crypto Pairs** | BTC-ETH, ETH-BTC | Cryptocurrency cross-pairs |
| **Stablecoins** | USDT-USD, USDC-USD | Stablecoin pairs |
| **DeFi Tokens** | UNI-USD, AAVE-USD | DeFi protocol tokens |

<Tip>
  Always check symbol status before placing orders. Symbols in "halt" or "maintenance" status will reject new orders.
</Tip>
