## Get Current Price

Retrieve the current market price for a specific trading pair.

### Endpoint
```http
GET /v1/market/price/{symbol}
```

### Parameters

<ParamField path="symbol" type="string" required>
  Trading pair symbol (e.g., "BTC-USD")
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Format: `Bearer YOUR_API_KEY`
</ParamField>

### Example Request

<RequestExample>
```bash cURL
curl -X GET "https://api.roxom.com/v1/market/price/BTC-USD" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://api.roxom.com/v1/market/price/BTC-USD',
    headers=headers
)

price_data = response.json()
print(f"Current price: {price_data['data']['price']}")
```

```javascript JavaScript
const response = await fetch('https://api.roxom.com/v1/market/price/BTC-USD', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const priceData = await response.json();
console.log('Current price:', priceData.data.price);
```
</RequestExample>

### Response

<ResponseExample>
```json Success
{
  "data": {
    "symbol": "BTC-USD",
    "price": "45127.68000000",
    "bid": "45125.50000000",
    "ask": "45129.86000000",
    "timestamp": "2024-01-15T10:30:00Z"
  },
  "timestamp": "2024-01-15T10:30:00Z",
  "request_id": "req_abc123"
}
```
</ResponseExample>

<ResponseField name="data" type="object">
  Price data for the requested trading pair

  <Expandable title="Price data properties">
    <ResponseField name="symbol" type="string">
      Trading pair symbol that was requested
    </ResponseField>
    
    <ResponseField name="price" type="string">
      Current market price with 8 decimal precision
    </ResponseField>
    
    <ResponseField name="bid" type="string">
      Highest current bid price
    </ResponseField>
    
    <ResponseField name="ask" type="string">
      Lowest current ask price
    </ResponseField>
    
    <ResponseField name="timestamp" type="string">
      ISO 8601 timestamp when price was recorded
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="timestamp" type="string">
  ISO 8601 timestamp when response was generated
</ResponseField>

<ResponseField name="request_id" type="string">
  Unique identifier for this API request
</ResponseField>
