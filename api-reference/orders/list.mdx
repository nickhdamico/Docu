## List Orders

Retrieve all orders for your account with powerful filtering and pagination options.

### Endpoint
```http
GET /v1/orders
```

### Parameters

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Format: `Bearer YOUR_API_KEY`
</ParamField>

<ParamField query="symbol" type="string">
  Filter by trading pair (e.g., "BTC-USD")
</ParamField>

<ParamField query="status" type="string">
  Filter by order status: "NEW", "PENDING", "FILLED", "CANCELED", "PARTIALLY_FILLED"
</ParamField>

<ParamField query="side" type="string">
  Filter by order side: "BUY" or "SELL"
</ParamField>

<ParamField query="type" type="string">
  Filter by order type: "MARKET", "LIMIT", "STOP", "STOP_LIMIT"
</ParamField>

<ParamField query="start_time" type="integer">
  Start timestamp for date range filter (Unix milliseconds)
</ParamField>

<ParamField query="end_time" type="integer">
  End timestamp for date range filter (Unix milliseconds)
</ParamField>

<ParamField query="limit" type="integer" default="100">
  Number of orders to return (max: 500)
</ParamField>

<ParamField query="cursor" type="string">
  Pagination cursor for retrieving next page
</ParamField>

```python Python
import requests
from datetime import datetime, timedelta

    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

# Get recent BTC orders

    'https://api.roxom.com/v1/orders',
    headers=headers,
    params={
        'symbol': 'BTC-USD',
        'status': 'FILLED',

orders = response.json()
for order in orders['data']:
    print(f"Order {order['order_id']}: {order['side']} {order['quantity']} @ {order['price']}")
```

## Pagination

Use cursor-based pagination to navigate through large result sets:

```python
def get_all_orders(symbol=None):
    """Get all orders using pagination."""
    all_orders = []
    cursor = None
    
    while True:
        params = {'limit': 500}
        if symbol:
            params['symbol'] = symbol
        if cursor:
            params['cursor'] = cursor
            
        response = requests.get(
            'https://api.roxom.com/v1/orders',
            headers=headers,
            params=params
        )
        
        data = response.json()
        all_orders.extend(data['data'])
        
        if not data['pagination']['has_more']:
            break
            
        cursor = data['pagination']['next_cursor']
    
    return all_orders
```

## Order Sorting

Orders are returned in reverse chronological order (newest first) by default. The sort order cannot be changed through the API.



## Export Orders

For bulk analysis, you can export order data:

```python
import csv
from datetime import datetime, timedelta

def export_orders_to_csv(filename, days=30):
    """Export orders to CSV file."""
    
    end_time = int(datetime.now().timestamp() * 1000)
    start_time = int((datetime.now() - timedelta(days=days)).timestamp() * 1000)
    
    all_orders = []
    cursor = None
    
    while True:
        params = {
            'start_time': start_time,
            'end_time': end_time,
            'limit': 500
        }
        if cursor:
            params['cursor'] = cursor
            
        response = requests.get(
            'https://api.roxom.com/v1/orders',
            headers=headers,
            params=params
        )
        
        data = response.json()
        all_orders.extend(data['data'])
        
        if not data['pagination']['has_more']:
            break
            
        cursor = data['pagination']['next_cursor']
    
    # Write to CSV
    with open(filename, 'w', newline='') as csvfile:
        fieldnames = ['order_id', 'symbol', 'side', 'type', 'quantity', 
                     'price', 'status', 'created_at', 'executed_price']
        writer = csv.DictWriter(csvfile, fieldnames=fieldnames)
        
        writer.writeheader()
        for order in all_orders:
            writer.writerow({
                'order_id': order['order_id'],
                'symbol': order['symbol'],
                'side': order['side'],
                'type': order['type'],
                'quantity': order['quantity'],
                'price': order.get('price', ''),
                'status': order['status'],
                'created_at': order['created_at'],
                'executed_price': order.get('executed_price', '')
            })
    
    print(f"Exported {len(all_orders)} orders to {filename}")
```
