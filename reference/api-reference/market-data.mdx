## Overview

The Market Data API provides access to real-time quotes, historical price data, orderbook information, and trading statistics for all supported instruments.

## Get Current Price

Retrieve the current market price for a specific trading pair.

### Endpoint
```http
GET /v1/market/price/{symbol}
```

### Parameters

<ParamField path="symbol" type="string" required>
  Trading pair symbol (e.g., "BTC-USD")
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Format: `Bearer YOUR_API_KEY`
</ParamField>

### Example Request

<RequestExample>
```bash cURL
curl -X GET "https://api.roxom.com/v1/market/price/BTC-USD" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://api.roxom.com/v1/market/price/BTC-USD',
    headers=headers
)

price_data = response.json()
print(f"Current price: {price_data['data']['price']}")
```

```javascript JavaScript
const response = await fetch('https://api.roxom.com/v1/market/price/BTC-USD', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const priceData = await response.json();
console.log('Current price:', priceData.data.price);
```
</RequestExample>

### Response

<ResponseExample>
```json Success
{
  "data": {
    "symbol": "BTC-USD",
    "price": "45127.68000000",
    "bid": "45125.50000000",
    "ask": "45129.86000000",
    "timestamp": "2024-01-15T10:30:00Z"
  },
  "timestamp": "2024-01-15T10:30:00Z",
  "request_id": "req_abc123"
}
```
</ResponseExample>

<ResponseField name="data" type="object">
  Price data for the requested trading pair

  <Expandable title="Price data properties">
    <ResponseField name="symbol" type="string">
      Trading pair symbol that was requested
    </ResponseField>
    
    <ResponseField name="price" type="string">
      Current market price with 8 decimal precision
    </ResponseField>
    
    <ResponseField name="bid" type="string">
      Highest current bid price
    </ResponseField>
    
    <ResponseField name="ask" type="string">
      Lowest current ask price
    </ResponseField>
    
    <ResponseField name="timestamp" type="string">
      ISO 8601 timestamp when price was recorded
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="timestamp" type="string">
  ISO 8601 timestamp when response was generated
</ResponseField>

<ResponseField name="request_id" type="string">
  Unique identifier for this API request
</ResponseField>

## Get Order Book

Retrieve current order book data for a trading pair.

### Endpoint
```http
GET /v1/market/orderbook/{symbol}
```

### Parameters

<ParamField path="symbol" type="string" required>
  Trading pair symbol (e.g., "BTC-USD")
</ParamField>

<ParamField query="depth" type="integer" default="20">
  Number of price levels to return. Range: 1-100
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Format: `Bearer YOUR_API_KEY`
</ParamField>

### Example Request

<RequestExample>
```bash cURL
curl -X GET "https://api.roxom.com/v1/market/orderbook/BTC-USD?depth=10" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```
</RequestExample>

### Response

<ResponseExample>
```json Success
{
  "data": {
    "symbol": "BTC-USD",
    "bids": [
      ["45125.50000000", "0.25000000"],
      ["45125.00000000", "0.50000000"],
      ["45124.50000000", "1.00000000"]
    ],
    "asks": [
      ["45129.86000000", "0.30000000"],
      ["45130.00000000", "0.75000000"],
      ["45130.50000000", "0.45000000"]
    ],
    "timestamp": "2024-01-15T10:30:00Z"
  }
}
```
</ResponseExample>

<ResponseField name="data" type="object">
  Order book data for the requested trading pair

  <Expandable title="Order book properties">
    <ResponseField name="symbol" type="string">
      Trading pair symbol that was requested
    </ResponseField>
    
    <ResponseField name="bids" type="array">
      Array of bid orders [price, quantity] sorted by price descending
    </ResponseField>
    
    <ResponseField name="asks" type="array">
      Array of ask orders [price, quantity] sorted by price ascending
    </ResponseField>
    
    <ResponseField name="timestamp" type="string">
      ISO 8601 timestamp when order book snapshot was taken
    </ResponseField>
  </Expandable>
</ResponseField>

## Get 24hr Statistics

Retrieve 24-hour trading statistics for one or all trading pairs.

### Endpoint
```http
GET /v1/market/stats/{symbol}
GET /v1/market/stats
```

### Parameters

<ParamField path="symbol" type="string">
  Trading pair symbol. Omit to get stats for all pairs
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication. Format: `Bearer YOUR_API_KEY`
</ParamField>

### Example Request

<RequestExample>
```bash Single Symbol
curl -X GET "https://api.roxom.com/v1/market/stats/BTC-USD" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```bash All Symbols
curl -X GET "https://api.roxom.com/v1/market/stats" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```
</RequestExample>

### Response

<ResponseExample>
```json Success
{
  "data": {
    "symbol": "BTC-USD",
    "open": "44950.00000000",
    "high": "45200.00000000",
    "low": "44800.00000000",
    "close": "45127.68000000",
    "volume": "1542.35000000",
    "quote_volume": "69847253.45000000",
    "price_change": "177.68000000",
    "price_change_percent": "0.3951",
    "count": 15634,
    "timestamp": "2024-01-15T10:30:00Z"
  }
}
```
</ResponseExample>

<ResponseField name="data" type="object">
  24-hour trading statistics

  <Expandable title="Statistics properties">
    <ResponseField name="symbol" type="string">
      Trading pair symbol
    </ResponseField>
    
    <ResponseField name="open" type="string">
      Opening price 24 hours ago

    <ResponseField name="high" type="string">

    <ResponseField name="low" type="string">

      Total volume traded in base asset

    <ResponseField name="quote_volume" type="string">
      Total volume traded in quote asset
    </ResponseField>
    
    <ResponseField name="price_change" type="string">

    <ResponseField name="price_change_percent" type="string">

    <ResponseField name="timestamp" type="string">

## Get Historical Klines

Retrieve historical kline/candlestick data.

### Endpoint
```http
GET /v1/market/klines/{symbol}
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `symbol` | string | Yes | Trading pair symbol |
| `interval` | string | Yes | Kline interval (1m, 5m, 15m, 30m, 1h, 4h, 1d, 1w) |
| `start_time` | integer | No | Start timestamp (Unix milliseconds) |
| `end_time` | integer | No | End timestamp (Unix milliseconds) |
| `limit` | integer | No | Number of klines (default: 500, max: 1000) |

### Example Request

```bash
curl -X GET "https://api.roxom.io/v1/market/klines/BTC-USD?interval=1h&limit=24" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

### Response

```json
{
  "data": [
    [
      1642248600000,           // Open time
      "44950.00000000",        // Open price
      "45050.00000000",        // High price
      "44900.00000000",        // Low price
      "45000.00000000",        // Close price
      "125.45000000",          // Volume
      1642252199999,           // Close time
      "5641253.45000000",      // Quote asset volume
      856,                     // Number of trades
      "62.75000000",           // Taker buy base asset volume
      "2820626.72500000"       // Taker buy quote asset volume
    ]
  ]
}
```

## Get Recent Trades

Retrieve recent trade executions.

### Endpoint
```http
GET /v1/market/trades/{symbol}
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `symbol` | string | Yes | Trading pair symbol |
| `limit` | integer | No | Number of trades (default: 100, max: 500) |

### Example Request

```bash
curl -X GET "https://api.roxom.io/v1/market/trades/BTC-USD?limit=50" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

### Response

```json
{
  "data": [
    {
      "id": "trade_12345",
      "price": "45127.68000000",
      "quantity": "0.25000000",
      "quote_quantity": "11281.92000000",
      "timestamp": "2024-01-15T10:30:00Z",
      "is_buyer_maker": false
    }
  ]
}
```

## Error Handling

Common error responses for market data endpoints:

### Symbol Not Found

```json
{
  "error": {
    "code": 404,
    "message": "Symbol not found",
    "details": "Trading pair 'INVALID-PAIR' does not exist"
  }
}
```

### Invalid Parameters

```json
{
  "error": {
    "code": 400,
    "message": "Invalid interval",
    "details": "Interval must be one of: 1m, 5m, 15m, 30m, 1h, 4h, 1d, 1w"
  }
}
```
