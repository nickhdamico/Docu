## List Orders

Retrieve orders for your account filtered by instrument type and optionally by symbol.

### Endpoint
```http
GET /api/v1/orders
```

### Parameters

<ParamField header="X-API-Key" type="string" required>
  API key for authentication
</ParamField>

<ParamField query="instType" type="string" required>
  Instrument type: "perpetual" or "spot"
</ParamField>

<ParamField query="symbol" type="string">
  Filter by trading symbol (e.g., "SP500-BTC", "GLD-BTC", "OIL-BTC")
</ParamField>

<ParamField query="unit" type="string">
  Display units: "sats" or "btc" (default: "sats")
</ParamField>

### Example Requests

<RequestExample>
```bash All Perpetual Orders
curl -X GET "https://api.roxom.com/api/v1/orders?instType=perpetual" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```bash Filtered by Symbol
curl -X GET "https://api.roxom.com/api/v1/orders?instType=perpetual&symbol=SP500-BTC" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```bash With BTC Units
curl -X GET "https://api.roxom.com/api/v1/orders?instType=perpetual&units=btc" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```python Python
import requests

headers = {
    'X-API-Key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
}

# Get all perpetual orders
response = requests.get(
    'https://api.roxom.com/api/v1/orders',
    headers=headers,
    params={
        'instType': 'perpetual'
    }
)

orders = response.json()
for order in orders['data']['orders']:
    print(f"Order {order['id']}: {order['side']} {order['quantity']} {order['symbol']}")
```

```javascript JavaScript
// Get orders for specific symbol
const response = await fetch('https://api.roxom.com/api/v1/orders?instType=perpetual&symbol=SP500-BTC', {
  headers: {
    'X-API-Key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
data.data.orders.forEach(order => {
  console.log(`${order.symbol}: ${order.side} ${order.quantity} @ ${order.price || 'MARKET'}`);
});
```
</RequestExample>

## Order Status Values

- **submitted**: Order submitted to matching engine and active
- **partially_filled**: Order partially executed, remainder still active  
- **filled**: Order completely executed
- **cancelled**: Order cancelled by user or system
- **rejected**: Order rejected due to validation errors

## Units Display

- **sats**: Quantities displayed in satoshis (default)
- **btc**: Quantities displayed in Bitcoin (decimal)
